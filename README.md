# 📊 Data Warehouse & Analytics Project

Welcome to the **Data Warehouse and Analytics Project** repository! 🚀  
This project demonstrates an end-to-end data warehousing and analytics solution — from raw data ingestion to generating actionable business insights.

It is designed as a **portfolio project** to showcase best practices in data engineering, data modeling, and analytics.

---

## 🏗️ Data Architecture

This project follows the **Medallion Architecture** approach:

### 🥉 Bronze Layer (Raw Data)
- Stores raw data directly from source systems  
- Data is ingested from CSV files into SQL Server  
- No transformations are applied  

### 🥈 Silver Layer (Cleaned Data)
- Data cleansing and standardization  
- Handling missing values and inconsistencies  
- Data normalization for consistency  

### 🥇 Gold Layer (Business-Ready Data)
- Data modeled into a **Star Schema**  
- Optimized for reporting and analytics  
- Contains fact and dimension tables

## ⚙️ Project Requirements

### 🏗️ Data Engineering (Data Warehouse Development)

**Objective:**  
Develop a modern data warehouse using SQL Server to consolidate sales data and enable analytics.

**Specifications:**
- Data Sources: ERP & CRM datasets (CSV format)  
- Data Quality: Clean and resolve inconsistencies before analysis  
- Integration: Combine datasets into a unified analytical model  
- Scope: Focus on latest dataset (no historization required)  
- Documentation: Provide clear data model documentation  

---

### 📊 Data Analysis (BI & Reporting)

**Objective:**  
Deliver SQL-based insights to support business decision-making.

**Key Focus Areas:**
- Customer Behavior  
- Product Performance  
- Sales Trends
